CLP Authentication Gateway Development

This domain (clp.us.ci) is dedicated to the integration and testing of transactional email services for our internal ASP.NET Core backend infrastructure.

Project Specifications

  • Framework: ASP.NET Core 9.0 (Web API)
  • Service Provider: Resend API (Free Tier)
  • Purpose: Handling automated system notifications and identity verification.

Planned Email Workflows

During the development phase, we are implementing the following transactional templates:

  • OTP_VERIFICATION: Sending 6-digit verification codes for user registration.
  • PASSWORD_RESET: Delivering secure, time-sensitive reset links.
  • SYSTEM_ALERT: Internal notifications regarding server status and API health.

Integration Status

Currently, the DNS records are managed via HE.net and the static documentation is hosted on Netlify. We are awaiting API activation to finalize our SMTP/API relay logic.

Note: This is a non-public environment. No marketing or bulk emails will be sent from this domain.
